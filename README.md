# P2HACKS2025 アピールシート

## プロダクト名
さんぽのプロ ~車は甘え~

## コンセプト
歩いてシンボルを育てる育成ゲーム
さんぽするきっかけを作る、シンボル育成アプリ 


## 対象ユーザ
運動不足の人（車を使っている人）

## 利用の流れ
1. アプリ開く
2. ユーザー登録
3. シンボルを設定
4. シンボルに向かって歩く
5. シンボルに着いたら、歩いた歩数分捧げる
6. シンボルがキラキラになる！

## 推しポイント
- レベルによって変わるシンボルの装飾
  - レベルを上げるために、歩きたくなる！~車は甘え~
- 歩数のカウントにpedometerを使用しているため、車でズルができない
- シンボルが**キラキラ**でかわいい

## 開発体制

### 役割分担
- はまさか：デザイン
- とがし：Flutter
- ごうだ：Flutter
- すずき：バックエンド

### 開発における工夫した点
- ある程度明確に役割を線引し、役割内で実装責任を持つ 
  - 多少のタスクのムラはあったとは思うが、比較的ちゃんと作業分担ができた
  - 事前に技術領域を共有していたので、ある程度お互いのカバーできる範囲を認識できていた
- Flutterの役割内でも、ベースの実装と整備で、ベルトコンベア式の作業を行った
  - 作業途中にコンフリクトが発生しにくく、スムーズな開発ができた
